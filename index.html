NumCpp

NumCpp is a high performance numerical C++ library that provides a simple Numpy/Matlab like interface

Download this project as a .zip file Download this project as a tar.gz file

Introduction

NumCpp is a numerical C++ library, which allows to write numerical algorithms in a similar way as in Numpy or Matlab. It uses features of the new C++11 standard simplifying both the implementation of NumCpp and the external API. Here is a simple example of its usage:

#include <numcpp.h>
using namespace numcpp;

int main()
{
  Vector<double> x = ones(16);
  auto y = fft(x);
  auto z = reshape(y,4,4);

  print(z);
}

Features

The features of NumCpp are

  • Simple interface similar to Numpy/Matlab
  • Multidimensional array object allowing for strided slicing
  • High performance using expression templates
  • Mixing of datatypes in expressions

Please have a look at the documentation page for an introduction into the concept of NumCpp.

License

The base functionallity of NumCpp is licensed under the LGPL. Some modules that require to link against software that is GPL licensed are licensed under the GPL as well. For instance the fft module uses internally the FFTW library, which is licensed under the GPL. If you plan to use NumCpp in a closed source software, please contact us to obtain a version of NumCpp where all GPL modules are replaced by GPL-free modules.
